# Nexterc
Nexterc is an ecommerc app built with next.js and tailwind css.

## Installition
```
// Clone the repository and install all dependencys
npm install
// or 
yarn install
```

## Run 
```
npm run dev
// or 
yarn dev
```

## Pages
- Home page (All products)
- product/productName (Product details)
- my-account (User account page)
- shoping-cart (Shopping cart page)
